# 验收测试说明

## 1) 快捷键响应与冲突

- 启动应用后按 `Win+T`（Electron 中 `Super+T`），确认 200ms 内弹出快速记录窗口。
- 在设置改为其它组合，若冲突会提示注册失败。

## 2) 10 万条数据查询

- 运行 `npm run test:perf:100k` 生成数据并测量查询。
- 目标：时间范围查询 + 状态筛选 < 1s。

## 3) 大模型离线缓存与降级

- 配置 API 后连续两次生成同一范围总结，第二次应命中缓存。
- 断网后再次生成：基础记录功能应保持可用，并显示降级提示。

## 4) 72 小时稳定性

- 运行 `npm run test:stability:72h`，同时执行定期新增/归档操作。
- 观察内存曲线是否持续上涨，是否出现崩溃。

## 5) 异常关机完整性

- 运行 `npm run test:poweroff` 做写入完整性模拟。
- 实机测试中可在写入期强制关机，重启后验证 JSONL 可恢复读取。
